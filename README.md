# GCMF Hierarchy Visualization

Interactive organizational hierarchy visualization for GCMF data. Renders complex reporting structures as a navigable, collapsible tree in the browser.

## Run

```bash
open gcmf_hierarch2y.html
```

Single HTML file, no dependencies. Opens in any modern browser.

## Tech

- Vanilla HTML/CSS/JavaScript
- Dynamic DOM rendering for hierarchical data
- Collapsible tree nodes for large org structures
